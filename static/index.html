TabDN Features Documentation Apply For Beta Samples

What can TabDN do?

Convert a csv into a table that lets users:
  • Sort
  • Filter
  • Create Pivot Tables
  • Add Columns
  • Compute Sums and more
  • Paginate
  • Rearrange Columns
TabDN serves tables:
  • Globally Accessible
  • Fast
  • Large
  • Customizable
  • Easy to Setup
  • Lightweight
Sort
Click any column header to sort.
Filter
Enter custom commands like 'HR+K+BB>1' or automatically add a filter by clicking on a cell. Then press the filter button to only display rows that match.
Add Columns
Easily calculate a new column.
Rearrange Columns
Move columns or remove unneeded columns.
Pivot Table
Compute pivot tables with any column.
Paginated
Instantly move to a different page.
Column Operations
Compute the sum, mean, max, or min of any or all columns using any filter.
Global Distribution
Servers around the world.
Customizable
Output as table, or set of cards, or create your own template.
Lightweight
10 kB of JS and only as much data as is displayed.
Easy to Setup
Upload csv, add 1 line of JS and 1 line of HTML.
Millions of Cells
Beta handles 10 Million cells (~25 MB of integers).
Fast
Most operations are completed before the user finishes their click.
